jQuery xFade Plugin

A jQuery plugin to fade/slide the elements inside the container based/extended from jQuery InnerFade by Torsten Baldes with the controls as addition.

Usage

The scripts

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4/jquery.min.js"></script>
<script type="text/javascript" src="jquery.xfade-1.0.min.js"></script>

Initialization

$(document).ready(function() {
  $('#container').xfade();
});

Available Options

Key Default Description
effect 'fade' Animation effect. Can be set to 'fade' or 'slide'.
speed 'normal' Transition speed in milliseconds or keywords 'slow', 'normal' or 'fast'.
timeout 2000 Delay between transitions in milliseconds.
order 'sequence' Items order. Can be set to 'sequence', 'random' or 'random-start'.
height 'auto' Container height.
containerClass false CSS class to added into container.
children false jQuery children selector.
onBefore false Callback that fires right before the transition. It receives the following arguments: current item index, last item index, current item, last item and items
